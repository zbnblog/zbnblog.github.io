安装Parrot操作系统到VMware虚拟机

本文完全原创，转载请标明原链接

# 注意：阅读本文需要一定的VMware经验和计算机知识。

①.安装Parrot到VMware

  1. 获得镜像。 我们要安装任何一个操作系统，首要的任务当然是获得那个系统的镜像， 所以需要前往Parrot OS的官网进行下载。
    【点击这里前往Parrot官网进行安装】
    或前往 www.parrotsec.org/download
    选择一个适合的版本进行下载（本文使用的是iso镜像），有时候可能安装速度极慢，则用IDM等工具解决此问题。
  2. 打开VMware。（我相信你们应该有办法解决VMware的许可证问题）
    选择“创建新的虚拟机”，选择“典型”，在下一步中选择“安装程序光盘映像文件”并选中Parrot操作系统映像文件。
  3. 创建虚拟机。下一步选择客户机操作系统为“Linux”，版本选“Debian 10.x 64位“，下一步，根据你的需要，改虚拟机名称和虚拟机位置。下一步根据需要，设置磁盘大小（不建议低于20GB）和虚拟磁盘的储存方式。下一步记得自定义系统硬件配置（Parrot系统要求内存不能低于4GB，否则有些功能将被停止），然后点击”完成“
  4. 启动虚拟机。并根据系统界面的指示进行安装（6.1版本请选择带”try/install“的一项进入系统，此时桌面上应有安装程序，点击进入才可以进入安装程序），设置磁盘分区时请选择”使用整个磁盘“并不要对磁盘进行加密，确保所有配置无误后开始安装。
  5. 重启。待安装完毕后按照系统提示进行重启，再次开机后应进入系统桌面（有密码的需要输入用户密码），版本≥6的不需要再次设置语言，因为安装的时候已经改变系统语言了。
  6. 设置sudo。此时你的系统无法使用sudo命令，因为没有设置root账户，需要开启终端输入sudo passwd root进行设置密码（设置root密码时，密码不会明文显示）。

②.优化系统与换源（进阶）

1. 换源可以使系统更新源的速度更加快捷

更新源的位置：/etc/apt/sources.list.d/parrot.list
换源：
  1. 官方更新源（速度一般）：
    deb http://deb.parrotsec.org/parrot stable main contrib non-free
  2. 清华大学镜像源（速度快）：
    deb https://mirrors.tuna.tsinghua.edu.cn/parrot/ parrot main contrib non-free
  3. 命令行：
备份源文件  
$ sudo cp /etc/apt/sources.list.d/parrot.list /etc/apt/sources.list.d/parrot.list.bak  
更改源文件  
$ sudo vim /etc/apt/sources.list.d/parrot.list

在vim终端编辑器先点击ESC确保不在写入的模式，然后输入:%d删除所有行，再点击A进入输入模式，输入要改变的源并再次点击ESC退出输入模式，输入:wq保存并退出
注：vim终端编辑器的操作详见
